# My Submission for Task 1: Static Landing Page (TaskFlow Pro)

Hi there! Thanks for reviewing my project. This is a clean and visually polished static landing page that I built from scratch for a product called **TaskFlow Pro**. 

I wanted to challenge myself, so I managed to build the entire page using only **pure HTML5 and CSS3**—there is absolutely zero JavaScript used here.

## ✨ Cool Stuff I Added:
* **Dark Mode Switch (No JS!):** I used the CSS checkbox trick to build a fully functional dark/light mode toggle. You can click the ☀️/🌙 icon in the navbar to test it out!
* **Smooth Animations:** The hero section has a nice fade-in animation when the page loads, and all buttons have smooth transition effects when you hover over them.
* **Modern UI Touches:** I added gradient text, background glow effects, and a glassmorphism blur effect on the sticky navbar.

---

## 📋 Checklist Verification

Here is how I covered all the project requirements:
- [x] **Sticky Navbar:** It stays fixed at the top when you scroll and includes 3 working section links (Features, About, Testimonials).
- [x] **Hero Section:** Has a bold headline, subtitle, and an eye-catching call-to-action button.
- [x] **2 Content Sections:** I designed a 3-card layout for "Features" and a 2-card layout for "Testimonials".
- [x] **Footer:** Added at the bottom with placeholder links for social media and privacy terms.
- [x] **Consistent Colors:** Everything is tied together using CSS variables so the color palette looks professional across light and dark modes.
- [x] **Fully Responsive:** I used CSS Grid and Media Queries. If you open it on a mobile phone (under 768px), the layout wraps perfectly into a single column.
- [x] **No Overlapping:** Added `box-sizing: border-box` to global styles and carefully managed margins and padding so nothing breaks.
- [x] **Typography:** Used Google Fonts (Plus Jakarta Sans) with different font-sizes and weights to maintain a clean text hierarchy.
